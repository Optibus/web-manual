Export Schedules

This button opens a sub-menu of its own:

Current Day

This downloads the current day schedule to a zip file.

All Days

This downloads the current week schedule to a zip file.

Compare

The Compare feature provides an extensive comparison between the currently open Schedule and any other of your choice.

Export Trips

A single trips Excel file is generated. It contains three pages, Trips, Places and Stops in the same format as the Operator's trip file required to create the Vehicles Gantt.
