Duty Homogeneity

Work-flow to Use Duty Homogeneity

  • There is currently one template
  • Use Load Template to set up the preference

You can repeat Load Template for additional instances of the template, but it does not make sense to do it.

Preference Overview

This preference allows you to set duty homogeneity by route sign with a penalty that increases with homogeneity. If there are no changeovers during the duty then duty homogeneity is the same as vehicle route sign homogeneity.

See Homogenic Vehicle Schedule in particular, Add Preference Dialog.
