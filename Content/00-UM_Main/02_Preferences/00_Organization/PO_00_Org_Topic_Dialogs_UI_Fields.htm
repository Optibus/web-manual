User Interface Fields

The user interface for most part, uses entry fields familiar to you from common Web applications. We will review them using the Depot Capacity example and other examples.

Look at the Depot Capacity preference again (Figure 1-2):

Stop is an example of a drop-down pick list:

End Day and Mid Day demonstrate switches that are toggled On or Off with a mouse click.

Branch ID is a simple text string.

You may be asked to choose one item from a list of radio buttons

The default or current choice is highlighted in blue.

Sometimes a simple choice opens a further dialog. In this example , if you choose Capacity by Vehicles, a further dialog opens:

You are required to enter the capacity by vehicle type. See Depot Capacity preference description for further details.

These vehicle types come from your data. They are not "hardwired" into the system. This preference will be modified to accommodate your own vehicle types. This is an excellent example of the flexibility of the system in meeting different Operator requirements.
