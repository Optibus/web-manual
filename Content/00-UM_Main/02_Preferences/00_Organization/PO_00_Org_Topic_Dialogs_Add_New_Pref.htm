Adding a New Preference

You will encounter one of two scenarios when adding a new preference:

In the first case, clicking the button offers a full data entry dialog. You can go ahead and enter the required values. Here is an example:

In the second case, you see an "Expert" type window. Here you will need to load a template with button instead of using the button:

See Adding a Preference by Loading a Template for a full example of how to load a template.

Further instances of the open preference may be added by repeated use of the Add Preference button.

Multiple preferences in an existing preference dialog are available when one instance has been added and you still see the Add Preferences button.

If for example, you open Duty Types under the Drivers topic, you will see that there is a set of preference values for each of 10 duty types.

Some preferences may only allow you to use Add Preferences more than once. These preference only allows one instance and thereafter will not display the Add Preference button. See for example, Driver Changeover Vehicle.
