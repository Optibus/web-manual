As a further example, with different field types, we look at the Pre\Post trip - General preference for the Vehicles topic. Here is the preference display:

Pre and Post Trip preferences user interface

Using Expert mode: See Expert Mode for the User Interface for a Topic Preference.

Items in a red box like this are mandatory for the template. If you do not enter them, the Save will work but the optimization with throw an "Optimization Failed" error with some technical debug information effectively telling you which fields were not filled.

The Pre trip and Post trip times use a standard hours:minutes format. Click the hours area. It is highlighted as active. We only require minutes here, so click the minutes area:

Enter 10 minutes as shown:

The highlighting disappears. It will not respond to non numeric characters; if you enter a value over 60 such as 75 it will just keep valid digits - in this case 7minutes

Change the Post trip time to 7 minutes.

The remaining radio buttons are obvious. As in the Quick Tour, we leave them as is.
