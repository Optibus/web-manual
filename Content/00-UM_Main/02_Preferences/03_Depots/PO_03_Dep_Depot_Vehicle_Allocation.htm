Use Load Template to set up the preference. Only one instance of this preference may be used. Two templates are available, one of which is an extension of the other.

Preference Overview

This preference sets rules for vehicle allocation to depots.

Advanced depot optimization

Purpose:

Determines vehicle depot circularity. Setting this requires limiting optimization iterations. Intended for multi-depot interurban operators.

Prerequisites: None.

Opening Dialog:

 

Points to note:

  • Vehicle depot circularity is set to Yes by default.
  • Max iterations: This sets the maximum number of optimization iterations for Depot allocation
  • Max none improvement iterations: If there is no improvement between iterations, stop the optimization after this number of iterations.

    • The default values shown are "best practice" to control optimization run-time. Change them in consultation with Professional Services.

  • If you disable circularity, the two iteration items are ignored.
