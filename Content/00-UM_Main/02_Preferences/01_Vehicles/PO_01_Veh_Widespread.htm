Widespread

Work-flow to Use Widespread

  • Use Add Preference to obtain a baseline dialog that sets Widespread only
  • Repeat Add Preference for additional instances of the baseline dialog
  • Use Load Template modify the preference for a wide variety of conditions
  • Repeat Load Template for additional instances of any template

Preference Overview

The Widespread preference places a time gap between trips of the Widespread time interval chosen. Additional conditions may be imposed using the supplied templates.

Add Preference Dialog

Purpose:

Place a time gap between trips of the Widespread time interval chosen. It will tend to alter KPIs for example increasing the number of vehicles and duties.

Prerequisites: None.

Opening Dialog:

Widespread - Basic dialog

Points to note:

  • This preference may be strict or may be made flexible by assigning a penalty
  • Using Edit Filters you can restrict the Widespread to a time interval
  • Using Edit Filters you can restrict it to one or more vehicle types

You can use Add Preferences to create additional instances of this preference. Use Edit Filters as indicated, to differentiate between them.

Widespread - by route

Prerequisites: None.

Opening Dialog:

Points to note:
