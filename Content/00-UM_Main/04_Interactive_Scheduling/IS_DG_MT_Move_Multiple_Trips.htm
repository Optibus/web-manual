Moving Multiple Trips to Another Duty

Moving multiple trips works the same way as moving a single trip with one difference: Instead of selecting a single trip, you select a range. We demonstrate the method using the following Drivers Gantt segment:

We will move the group of trips shown in driver block 203.

To move a group of trips:

Click the first trip in block 203.

Shift-click the last trip in block 203.

The information box confirms your selection:

Click the green Move button.

The time range is highlighted and available slots are shown in the same manner as for moving a single trip.

Proceed in the same manner as for a single trip.

This method can only be used for blocks of contiguous trips.

Moving multiple trips is a useful device for "cutting" a block of drips and using them to create a new duty. See Moving Trips to a New Duty.
