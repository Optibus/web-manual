Moving Trips to a New Duty

We will use the example from Moving Multiple Trips to Another Duty to create a new duty.

To create a new duty in the Drivers Gantt:

Scroll down to the bottom of the Drivers Gantt:

The blue + sign below the last duty (458) is a place-holder for a new duty.

Double click the slot in the new duty block. A new duty is generated with a temporary duty ID as shown:
