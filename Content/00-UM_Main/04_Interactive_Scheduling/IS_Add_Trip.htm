Using Add Trip

Add Trip is available from the top of the Gantt display. To demonstrate the use of this feature, we start out with a segment of the Vehicles Gantt from the schedule we have used throughout this manual:

We will add an inter-urban trip for sign 192 at 10:35.

To add a trip:

Click the plus sign button at the top left of the Stack area:

It opens up the following dialog:

Add Trip dialog

Fill out the dialog, setting the direction, start time and trip duration.

Click Add: The new trip in the editing stack ready for assignment:

Double click the new trip in the editing stack.

From this point, proceed as in Moving a trip to another Duty and Using the Editing Stack.
